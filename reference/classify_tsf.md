# classify_tsf

Classify raster time since fire into bins

## Usage

``` r
classify_tsf(raster, mask = NULL)
```

## Arguments

- raster:

  Raster (terra).

- mask:

  Optional raster mask layer.

## Value

Raster layer with time since fire classes.
