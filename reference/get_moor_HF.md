# Calculate moorland FDI humidity factor

Calculate moorland FDI humidity factor

## Usage

``` r
get_moor_HF(temp, humid)
```

## Arguments

- temp:

  Temperature in degrees C

- humid:

  Humidity percentage

## Value

Moorland FDI Humidity Factor

## Examples

``` r
get_moor_HF(12,87)
#> [1] 24.99687
```
