# Get Evapotranspiration value for given Temperature and Soil Dryness Index

Get Evapotranspiration value for given Temperature and Soil Dryness
Index

## Usage

``` r
get_ET(temp, SDI)
```

## Arguments

- temp:

  Temperature in Degrees C

- SDI:

  Mount's Soil Dryness Index

## Value

Evapotranspiration value

## Examples

``` r
get_ET(15,221)
#> [1] 0
```
