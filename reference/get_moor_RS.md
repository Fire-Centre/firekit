# Calculate Moorland FDI rate of spread

Calculate Moorland FDI rate of spread

## Usage

``` r
get_moor_RS(FM, wind, age)
```

## Arguments

- FM:

  Fuel moisture as calculated by get_moor_FM

- wind:

  Wind speed

- age:

  Age of fuel (time since fire)

## Value

Rate of Spread

## Examples

``` r
get_moor_RS(18,10,25)
#> [1] 8.097372
```
