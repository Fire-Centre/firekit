# firekit_update

Sets or pulls stored GitHub credentials and initiates updates of firekit
package.

## Usage

``` r
firekit_update(dependencies = NA, force = FALSE)
```

## Arguments

- dependencies:

  Option to list dependencies to install.

- force:

  Option to force install even if currently running.

## Value

Installed packages

## Examples

``` r
if (FALSE) { # \dontrun{
firekit_update(force = TRUE)
} # }
```
