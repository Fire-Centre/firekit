# Construct line vector from a point over a given distance

Construct line vector from a point over a given distance

## Usage

``` r
line_vector(point, angle, distance)
```

## Arguments

- point:

  sf object with a point

- angle:

  Compass angle in degrees

- distance:

  Distance in metres

## Value

sf LineString object of vector

## Examples

``` r
if (FALSE) { # \dontrun{

line_vector(point,15,50)
} # }
```
