# fc_remove

Removes an existing database object from the `firecentre` PostgreSQL
database.

## Usage

``` r
fc_remove(tbl, db = "firecentre")
```

## Arguments

- tbl:

  Database schema and table name: Must be in `schema.table` syntax.

- db:

  Database name or path/to/database/object.

## Value

Data exported to database.
