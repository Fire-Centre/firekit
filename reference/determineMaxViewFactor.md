# Determine Maximum View Factor

Determine Maximum View Factor

## Usage

``` r
determineMaxViewFactor(dist, Lf, site_slope, h, Wf)
```

## Arguments

- dist:

  Distance (metres)

- Lf:

  Flame length

- site_slope:

  Slope of site (degrees)

- h:

  Elevation of receiver (m)

- Wf:

  Flame width

## Value

MaxViewFactor

## Examples

``` r
determineMaxViewFactor(20,5,5,2,100)
#> [1] 0.1206
```
