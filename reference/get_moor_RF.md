# Calculate Moorland FDI Rainfall Factor

Calculate Moorland FDI Rainfall Factor

## Usage

``` r
get_moor_RF(rainHours, rainAmount)
```

## Arguments

- rainHours:

  Number of hours over which rainfall fell

- rainAmount:

  Amount of rain in mm

## Value

Moorland FDI Rainfall Factor

## Examples

``` r
get_moor_RF(24,20)
#> [1] 8
```
